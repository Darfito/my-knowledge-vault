## Implementation (11.2–11.5)

### `worker--dispatch` SKILL.md — completely rewritten

Core changes from the old Anthropic SDK version:

| Old | New |
|---|---|
| `anthropic.AsyncAnthropic(api_key=WORKER_API_KEY)` | `claude -p` subprocess via `asyncio.create_subprocess_exec` |
| Required `WORKER_API_KEY` env var | No env vars required (subscription auto) |
| Per-worker cost event (Sonnet pricing) | One batch event, `biller: "subscription"`, `costCents: 0` |
| No concurrency cap | `asyncio.Semaphore(5)` |
| No rate-limit detection | `is_rate_limited()` parses 6 stderr patterns |
| No fallback | `USE_API_KEY=true` + `WORKER_API_KEY` restores SDK path |
